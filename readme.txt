== Description ==

This GamiPress add-on adds triggers for publishing and commenting on custom post types.

**Note:** You will need to install the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress")&trade; (version 1.4 or higher) to use the GamiPress Custom Post Type Addon. The Plugin was tested up to GamiPress 1.9.4.

[Get the GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress").

This plugin was originally developed for the [Academy of International Cooperation](https://www.giz.de/akademie/en/html/index.html) of [GIZ](https://www.giz.de).

== Installation ==

1. Upload, activate and configure the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress") to WordPress.
3. Upload 'Custom Post Type Add-On' to the '/wp-content/plugins/' directory
4. Activate the plugin through the 'Plugins' menu in WordPress
